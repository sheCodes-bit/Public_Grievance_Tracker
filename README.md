## Public Grievance Tracker

The **Public Grievance Tracker** is a full-stack web application that provides a transparent and efficient platform for citizens to report and track public issues. It helps bridge the gap between the public and authorities by enabling faster complaint registration, monitoring, and resolution.

### 🔹 Introduction

* A web-based system for reporting civic issues like infrastructure, health, education, and sanitation
* Connects citizens directly with authorities for faster issue resolution
* Ensures transparency through real-time status tracking of complaints
* Provides a centralized system to manage all public grievances

###  Key Features

* User registration and secure login using JWT authentication
* Submit grievances with title, description, category, and priority
* Track complaint status: Pending, In Progress, Resolved
* Admin dashboard to manage, update, and resolve complaints
* Upvote and manage public issues for better visibility
* Responsive and user-friendly interface

###  Tech Stack

* Frontend: HTML, CSS, JavaScript, jQuery, Bootstrap
* Backend: Node.js, Express.js
* Database: MongoDB (Mongoose)
* Authentication: JSON Web Token (JWT)

###  Purpose

* Improve civic engagement between citizens and authorities
* Increase transparency in public complaint handling
* Provide a fast and organized grievance resolution system
* Empower users to raise and track issues effectively
